# CHG-SIG: ISMS Significant Change (Category 2)

## Quick Reference
- **SLA:** 5-10 days
- **Team:** Internal-Ops (INT-)
- **Project:** Significant Changes

## Required Labels
- `Type: improvement`
- `Flag: compliance`
- `Flag: security` (if security control affected)
- `Layer: [affected layers]`
